## Lineage

### Paint-AR (2019)

> **The world's first holographic storytelling app.**

- Released 2019 on Apple App Store
- Featured on CNN with artworks by **Beeple & Richard Prince**
- Foretold spatial computing revolution before Apple Vision Pro

**What it did:**
- Paint in 3D, add holograms, sculpt
- Join featured artists, record video, share
- Remix your world, tell trippy tales

**GitHub:** [imclab/Paint-AR_app_Unity](https://github.com/imclab/Paint-AR_app_Unity)
**App Store:** [Paint-AR](https://apps.apple.com/ai/app/paint-ar/id1347083887)
**Website:** [paint-ar.com](http://paint-ar.com/)

### Open Source Philosophy

> "Give back to the community that has given so much to us."

Open doors to more XR creators, Unity/Unreal developers who share passions for:
- Free expression
- Powerful collaborative tools
- Next generation of mixed reality storytellers, artists & activists

## Technical Foundation (From Paint-AR)

### Core Systems

| System | Description |
|--------|-------------|
| **Brushes** | Tube, Flat, Square, Particles, Grab |
| **Holograms** | Scriptable Objects, Categories, Pinch/Zoom/Transform |
| **Worlds** | Terrains, Portals, Skyboxes |
| **FX** | Post-Processing, Sculpt, Camera Effects |
| **Save/Share** | OBJ export, Scene saving, ReplayKit |

### Key Scripts (Paint-AR 1.0)

- `BrushManager.cs` - Drawing system
- `DLine.cs`, `PinchDraw.cs` - Gesture painting
- `MeshExporter.cs` - Export to OBJ
- `PopupManager.cs` - UI system

### Versions

| Version | Status | Notes |
|---------|--------|-------|
| 1.0 | Released 2019 | Original Unity project |
| 1.5 | Partial | URP upgrade started |
| 2.0 | In progress | VisionOS + ARKit 5-6 migration |
