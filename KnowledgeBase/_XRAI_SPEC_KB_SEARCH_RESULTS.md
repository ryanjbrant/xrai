## XRAI Spec Context

The XRAI Specification & Architectural Plan covers:
- **Holographic HUD** ("Jarvis" Interface): Floating non-intrusive UI in 3D space with FLIGHT mode (exploration/gesture controls) and YOLO mode (camera segmentation + Gemini suggestions)
- **AI Orchestration Engine**: "Hey X-ray" trigger, gemini-2.5-flash-lite for fast response, gemini-3-pro-preview with 32768 thinking budget for complex tasks, gemini-2.5-flash-preview-tts for voice output (24kHz PCM)
- **Collaborative Telepresence**: WebRTC P2P data channels, shared "Persistence World" with holographic cursors/code
- **Persistence**: GDrive + GitHub export of XRAI state, custom interfaces, code checkpoints
- **Goal**: Maximizing collaborative creativity with highly flexible, super fast, modular, scalable agentic coding swarm, generating intuitive broad use case vibe coding interface & spatial intelligence engine
