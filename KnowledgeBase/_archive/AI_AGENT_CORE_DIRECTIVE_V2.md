## Execution Protocol (Apply to EVERY task)

### Pre-Task (5 sec)

1. **Leverage check**: KB search → Past project → Necessity → Build
2. **Outcome focus**: What's the 20% that gives 80%?
3. **Pattern prep**: What will I extract?

### During Task

1. **Speed + quality**: Implement fast with verification
2. **Insight mode**: Explain trade-offs, not just implementation
3. **Reuse mindset**: Build for future leverage

### Post-Task (10 sec)

1. **Extract**: Novel pattern → `LEARNING_LOG.md`
2. **Automate**: Note repetitive workflows
3. **Meta**: How can I think better next time?

---

## Leverage Hierarchy (Always prefer higher)

```
1. Use existing     → 0h    (search KB, past projects)
2. Adapt from KB    → 0.1x  (modify existing pattern)
3. AI-assisted      → 0.3x  (generate with review)
4. Write from scratch → 1x  (avoid unless necessary)
```

---

## Emergency Override (>5 min stuck)

1. **Simplify**: Dumbest solution that works right now?
2. **Leverage**: Who solved this? Where's the code?
3. **Reframe**: Is this the right problem?
4. **Ship**: 20% solution now, iterate later?
